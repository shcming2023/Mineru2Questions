## 11 - 题目 2

例②  $A, B$  两地相距  $50 \mathrm{~km}$ , 甲骑自行车从  $A$  地前往  $B$  地,  $1 \mathrm{~h} 30 \mathrm{~min}$  后乙骑摩托车也从  $A$  地前往  $B$  地, 结果乙比甲先到  $1 \mathrm{~h}$ . 已知乙的速度是甲的 2.5 倍, 求甲、乙两人的速度.
分析 本题的相等关系：甲的行进时间  $= \text{乙}$  的行进时间  $+1 + 1\frac{1}{2}$ .
解 设甲的速度为  $x \mathrm{~km} / \mathrm{h}$ , 则乙的速度为  $2.5x \mathrm{~km} / \mathrm{h}$ .
根据题意，可列方程  $\frac{50}{x} = \frac{50}{2.5x} + 1 + 1\frac{1}{2}$ . 解得  $x = 12$ 经检验，  $x = 12$  是所列方程的根且符合题意 乙的速度  $= 2.5x = 30\mathrm{km / h}$ 答：甲的速度是  $12\mathrm{km / h}$  ，乙的速度是  $30\mathrm{km / h}$

**答案:** 甲的速度是12km/h，乙的速度是30km/h

## 21.4 - 题目 3

如果已知一元二次方程  $ax^2 + bx + c = 0 (a \neq 0)$  的两个实数根为  $x_1, x_2$ ，就有  $ax_1^2 + bx_1 + c = 0, ax_2^2 + bx_2 + c = 0$ .
疑难分析
例① 求一个一元二次方程，它的两根分别是方程  $5x^{2} + 2x - 3 = 0$  各根的负倒数.
解 设方程  $5x^{2} + 2x - 3 = 0$  的两根分别为  $x_{1}, x_{2}$ .
由一元二次方程根与系数的关系，得  $x_{1} + x_{2} = -\frac{2}{5}, x_{1} \cdot x_{2} = -\frac{3}{5}$ .
设所求方程为  $y^{2} + py + q = 0$  ，它的两根为  $y_{1}, y_{2}$
由已知，得  $y_{1} = -\frac{1}{x_{1}}, y_{2} = -\frac{1}{x_{2}}.$
由一元二次方程根与系数的关系，得
$$
p = - \left(y _ {1} + y _ {2}\right) = - \left(- \frac {1}{x _ {1}} - \frac {1}{x _ {2}}\right) = \frac {1}{x _ {1}} + \frac {1}{x _ {2}} = \frac {x _ {1} + x _ {2}}{x _ {1} x _ {2}} = \frac {- \frac {2}{5}}{- \frac {3}{5}} = \frac {2}{3},
$$
$$
q = y _ {1} \cdot y _ {2} = \left(- \frac {1}{x _ {1}}\right) \cdot \left(- \frac {1}{x _ {2}}\right) = \frac {1}{x _ {1} x _ {2}} = - \frac {5}{3}.
$$
故所求方程为  $y^{2} + \frac{2}{3} y - \frac{5}{3} = 0$ ，即  $3y^{2} + 2y - 5 = 0$
说明 所求方程中的未知数与已知方程中的未知数要用不同的字母加以区别. 同时“以两个数  $x_{1}$ ,  $x_{2}$  为根的一元二次方程是  $x^{2} - (x_{1} + x_{2})x + x_{1} \cdot x_{2} = 0$ ”可以用这种语言形式记忆“ $x^{2}$  一和  $x$  十积  $= 0$ ”，或“减和加积”，此处的一次项系数最容易出现符号上的错误.
例② 已知三个不同的实数  $a, b, c$  满足  $a - b + c = 3$ , 关于  $x$  的方程  $x^2 + ax + 1 = 0$  和  $x^2 + bx + c = 0$  有一个相同的实数根, 关于  $x$  的方程  $x^2 + x + a = 0$  和  $x^2 + cx + b = 0$  也有一个相同的实数根. 求  $a, b, c$  的值.
分析 将题设中所给的四个方程编号为①、②、③、④. 设  $x_{1}$  是方程①和方程②的一个相同的实
数根， $x_{2}$  是方程③和方程④的一个相同的实数根，得到关于  $x_{1}$  与  $x_{2}$  的解析式，进而求出  $a$  的值，再求出  $b, c$  的值.
解 依次将题设中所给的四个方程编号为①、②、③、④.
设  $x_{1}$  是方程①和方程②的一个相同的实数根，则
$$
\left\{ \begin{array}{l} x _ {1} ^ {2} + a x _ {1} + 1 = 0, \\ x _ {1} ^ {2} + b x _ {1} + c = 0. \end{array} \right.
$$
两式相减，解得  $x_{1} = \frac{c - 1}{a - b}$
设  $x_{2}$  是方程③和方程④的一个相同的实数根，则
$$
\left\{ \begin{array}{l} x _ {2} ^ {2} + x _ {2} + a = 0, \\ x _ {2} ^ {2} + c x _ {2} + b = 0. \end{array} \right.
$$
两式相减，解得  $x_{2} = \frac{a - b}{c - 1}$
所以  $x_{1}x_{2} = 1$
又因为方程①的两根之积等于1,于是  $x_{2}$  也是方程①的根，即  $x_{2}^{2} + ax_{2} + 1 = 0.$
又因为  $x_{2}^{2} + x_{2} + a = 0$  ，两式相减，得  $(a - 1)x_{2} = a - 1.$
若  $a = 1$  ，则方程  $①$  无实根，所以  $a\neq 1$  ，故  $x_{2} = 1$
将  $x = 1$  代入方程③和方程④，得  $a = -2, b + c = -1$ .
又由  $a - b + c = 3$  ，解得  $b = -3, c = 2$ .
综上所述，  $a = -2$  ，  $b = -3$  ，  $c = 2$
说明 本题考查了一元二次方程  $ax^2 + bx + c = 0 (a \neq 0, a, b, c$  为常数)的根，同时考查了从结论思考问题的方法和代数式的变形能力.
